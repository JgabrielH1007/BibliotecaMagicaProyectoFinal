# BibliotecaMagicaProyectoFinal
Segundo proyecto de estructuras de datos.
PASOS PARA COMPILACION:
1. Clonar el repositorio, copiando el link del mismo e ingresado el comando "git clone "link repositorio""
2. Posicionarte en la carpeta del proyecto, verificar que se encuentre el .jar.
3. Abrir terminal o CMD en la ubicación del .jar.
4. Ingresar el comando: "java -jar nombredelarchivo.jar"
5. Ya habrás ejecutado el programa.
